Rails Migration Examples for Product Model

1. Creating the Products Table

To create a products table, you can use the following Rails migration:


class CreateProducts < ActiveRecord::Migration[6.1]
  def change
    create_table :products do |t|
      t.string :name
      t.decimal :price, precision: 10, scale: 2
      t.integer :quantity
      t.string :category

      t.timestamps
    end
  end
end

2. Inserting Data into the Products Table

You can insert data using Rails seeds or the Rails console. Here's an example for db/seeds.rb:


# db/seeds.rb
Product.create(name: 'Laptop', price: 1000.50, quantity: 10, category: 'Electronics')
Product.create(name: 'Phone', price: 500.99, quantity: 30, category: 'Electronics')

To load the seed data, run rails db:seed.

3. Deleting Data from the Products Table

You can delete data using the Rails console or controller. Here's an example:


# Deleting a product with ID 1
product = Product.find(1)
product.destroy

Or, delete data based on conditions:


# Deleting all products in a specific category
Product.where(category: 'Electronics').destroy_all

4. Updating Data in the Products Table

You can update existing data using the update method:


# Updating the price of a product with ID 1
product = Product.find(1)
product.update(price: 1200.75)

Alternatively, use an SQL query inside a migration:


class UpdateProductPrice < ActiveRecord::Migration[6.1]
  def up
    execute <<-SQL
      UPDATE products
      SET price = 1200.75
      WHERE id = 1;
    SQL
  end
end

5. Dropping the Products Table

If you want to drop the products table, use the following migration:


class DropProducts < ActiveRecord::Migration[6.1]
  def change
    drop_table :products
  end
end
